# Books Scraper (Scrapy)

A simple **Scrapy spider** that crawls book listings from **Books to Scrape** and extracts book titles and links from selected categories.

The scraped data is stored in a **Pandas DataFrame** and exported to an **Excel file**.

## Features

- Scrapes books from multiple categories
- Extracts:
  - Book title
  - Book link
- Stores data in a **Pandas DataFrame**
- Exports results to an **Excel file (`books.xlsx`)**

## Target Website

https://books.toscrape.com

Categories scraped:
- Travel
- Mystery
